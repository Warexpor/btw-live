name: btw-session-use
description: Switch active /btw-vc voice session by name or id. Use for /btw-session-use.
argument-hint: name-or-id
---

Take the session name or id from the user. Call `btw_session_use` with `{ "id_or_name": "..." }`.  
Confirm which session is active and its profile. If missing, list with `btw_session_list`.
